# Supermarket Sales Analysis — Myanmar 2019
Excel analysis of 1,000 supermarket transactions across 3 branches in Myanmar — covering branch performance, customer demographics, payment methods and product line insights.

## Tools Used
- Microsoft Excel
- Power BI (coming soon)

## Files
- SuperMarket_Analysis.xlsx — Full analysis and dashboard
- Supermarket_Sales_Report.docx — Full written report

## Kaggle Notebook
https://www.kaggle.com/code/liya7862003/supermarket-sales-analysis-myanmar-2019/

## Key Findings
- Total Revenue: $322,967
- Top Branch: Giza (34.24% of sales)
- Top Product Line: Food & Beverages (17.38%)
- Female customers drive 60.28% of total sales
